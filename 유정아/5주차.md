## 2.

고차함수를 활용해 아래 배열에서 이름이 3글자 이하인 이름만 뽑아 "님"을 이름 끝에 추가한 뒤 출력하세요.

```swift
let names = ["이상해씨", "꼬북이", "어니부기", "피카츄", "가라도스", "피츄", "뮤"]

let newNames: [String] = names.filter { $0.count <= 3 }.map { $0 + "님" }

print(newNames)
```

<br>

> 출력결과:
> 

```swift
["꼬북이님", "피카츄님", "피츄님", "뮤님"]
```
